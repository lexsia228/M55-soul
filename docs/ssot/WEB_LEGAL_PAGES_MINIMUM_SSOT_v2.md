# WEB Legal Pages Minimum SSOT v2（Stripe審査向け / 返金不可の明示を統合）
Freeze date: 2026-02-26 (JST)

目的: Stripe審査（およびカード会社の一般的審査）で「商売の実体」「責任の所在」「問い合わせ可能性」を即理解できる状態を、最小ページ群で確保する。

## 0. 上位制約（M55共通）
- 煽り文言（おすすめ/最適/成功/当たる/診断/断言）を入れない。
- 個人情報の公開は最小化し、メール窓口は監視できる実在アドレスに固定する。
  - 住所/電話は「請求があれば遅滞なく提供します」方式を許容。
- 返金ポリシーは「事実のみ・短文」で統一し、決済事業者の係争（チャージバック）に耐える表現を採用する。

## 1. 必須ページ（最低3枚 + 特商法）
A. 特定商取引法に基づく表記
- path: /legal/tokushoho
- 最小要件:
  - 販売価格（購入画面に表示）
  - 支払方法（Stripe / クレジットカード）
  - 引渡時期（決済完了後、即時提供）
  - 返品・キャンセル（下記「返金不可」条項を必ず含める）

B. プライバシーポリシー
- path: /legal/privacy
- 最小要件:
  - 収集する情報（Clerkのアカウント識別子、決済識別子、アクセスログ等）
  - 利用目的（提供、本人確認、決済、問い合わせ対応、改善）
  - 第三者提供（Stripe / Supabase / Vercel 等、必要最小の委託）
  - 問い合わせ窓口メール

C. 利用規約
- path: /legal/terms
- 最小要件:
  - サービス概要（デジタルコンテンツ提供）
  - 禁止事項（不正アクセス、転売、過剰なスクレイピング等）
  - 免責（事実の断定を避け、一般的な範囲で）
  - 返金不可（下記条項を必ず含める）

D. サポート
- path: /support
- 最小要件:
  - 連絡先メール（監視可能）
  - 返信目安（例: 2–5営業日）※断言は避ける

## 2. 返金不可（No Refund）条項（必須）
特商法（返品・キャンセル欄）と利用規約の双方に、同一トーンで必ず記載する。

推奨文（短文・断言過剰を避ける）:
- 「デジタルコンテンツの性質上、購入確定後のキャンセルおよび返金には原則として応じられません。」
- 追記（任意・法令整合のため）:
  - 「法令上必要な場合を除きます。」

## 3. 共通フッター（到達性）
- app/layout.tsx（または共通Shell）のフッターに、以下4リンクを常時露出（ログイン不要）。
  - /legal/tokushoho
  - /legal/privacy
  - /legal/terms
  - /support
- 視認性は確保しつつ主張しない（小さめ文字 / underline / 低めopacity）。
